# Installation Guide: Sokker++

Follow these steps to install the **Sokker++** extension on your Chrome, Edge, or Brave browser. This guide is designed for manually installing the extension from the downloaded `sokker-plus-plus.zip` file.

---

## üöÄ Step 1: Prepare the File

1.  **Download** the `sokker-plus-plus.zip` file (usually saved in your *Downloads* folder).
2.  **Unzip** the file:
    *   Right-click on `sokker-plus-plus.zip`.
    *   Select **"Extract All..."** or **"Extract Here"**.
    *   This will create a folder (which often contains another folder named `dist` or the extension files). **Remember where this folder is located.**

---

## üåê Option A: Google Chrome or Brave

1.  Open Google Chrome or Brave.
2.  In the address bar, type: `chrome://extensions/` or `brave://extensions/` and press **Enter**.
3.  In the top right corner, **enable** the **"Developer mode"** toggle.
4.  Three new buttons will appear on the left. Click the first one: **"Load unpacked"**.
5.  A folder selection window will open.
    *   Navigate to the folder where you unzipped the file.
    *   **Important:** You must select the folder containing the `manifest.json` file. If unzipping created a structure like `sokker-plus-plus/dist`, select the `dist` folder.
6.  Click **"Select Folder"** (or "Select").
7.  Done! The Sokker++ extension should now appear active in your list.

---

## üåä Option B: Microsoft Edge

1.  Open Microsoft Edge.
2.  In the address bar, type: `edge://extensions/` and press **Enter**.
3.  In the left menu (or bottom left), enable the **"Developer mode"** toggle.
4.  New buttons will appear. Click on **"Load unpacked"**.
5.  A folder selection window will open.
    *   Navigate to the folder where you unzipped the file.
    *   **Important:** You must select the folder containing the `manifest.json` file. If unzipping created a structure like `sokker-plus-plus/dist`, select the `dist` folder.
6.  Click **"Select Folder"** (or "Select").
7.  Done! The Sokker++ extension should now appear active.

---

## ‚úÖ Verification

1.  Go to [Sokker.org](https://sokker.org).
2.  Navigate to the **Squad** or **Players** page.
3.  You should see skill improvement arrows and charts when hovering over skills.

---

*Note: When restarting the browser, you may see a warning about "extensions in developer mode". This is normal for manually installed extensions and is 100% safe.*
